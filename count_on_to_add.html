Count On to Add

Summary

Counting on is a strategy where we start with the bigger number and count up. For example, to add 3 + 2, we start at 3 and count up: 4, 5. So, 3 + 2 = 5.

Examples

  • 5 + 2 ‚Üí Start at 5: count 6, 7 ‚Üí Answer: 7
  • 6 + 3 ‚Üí Start at 6: count 7, 8, 9 ‚Üí Answer: 9
  • 2 + 4 ‚Üí Start at 4: count 5, 6, 7, 8 ‚Üí Answer: 6
