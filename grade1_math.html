🎯 Grade 1 Mathematics Content Outline

🧮 1. Numbers and Number Sense

  • Count forward and backward (0–120)
  • Identify, write, and represent numbers (0–120)
  • Count objects and represent numbers with models
  • Understand place value (tens and ones)
  • Compare numbers using >, <, =

➕ 2. Addition and Subtraction

  • Addition Strategies up to 10
  • Count on to add
  • Use doubles and near doubles
  • Add in any order (commutative property)
  • Use number lines and number bonds
  • Count back
  • Use fact families
  • Relate subtraction to addition
  • Fluency
  • Add and subtract within 10 fluently
  • Add and subtract within 20 using strategies

🔟 3. Operations and Algebraic Thinking

  • Understand the meaning of the equal sign
  • Determine if equations are true or false
  • Solve word problems (addition and subtraction up to 20)
  • Represent problems with drawings and equations

📏 4. Measurement and Data

  • Compare lengths using non-standard units (e.g., blocks, paper clips)
  • Order three objects by length
  • Tell and write time to the hour and half hour
  • Collect, sort, and represent data on bar graphs and picture graphs

🔷 5. Geometry

  • Identify and name 2D shapes: circle, square, triangle, rectangle
  • Identify and name 3D shapes: sphere, cube, cone, cylinder
  • Compose and decompose shapes
  • Create shapes with given attributes (e.g., number of sides)

🎲 6. Patterns and Sorting

  • Recognize, extend, and create repeating patterns
  • Sort and classify objects by attributes (shape, size, color)

💡 7. Reasoning and Problem Solving

  • Use logical thinking to solve puzzles
  • Explain mathematical thinking using models and drawings
  • Practice word problems in real-life situations
